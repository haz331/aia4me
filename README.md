# AIA 4 Me

Your personal AI assistant with voice, chat, and wake word. Powered by your own OpenAI API key.

## Live App
https://ai4meassistant.base44.app

## Features
- Voice input with auto-silence detection
- Text-to-speech responses  
- Wake word activation
- Streaming chat with GPT-4o
- API key stored locally on device only
- Full PWA installable on any device
